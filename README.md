# neovim-cheatsheet
Just a cheatsheet for my neovim config
# LSP stuff

| Key      | Function |
| ----------- | ----------- |
| gD       | Goto declaration       |
| gd   | Goto definition        |
| gr | Show references |
| gi | Goto implementation |
| K | Hover |
| C-k | Signature help |
| space wa | Add workspace folder |
| space wr | Remove workspace folder |
| space wl | List workspace folders |
| space D | Goto type definition |
| space rn | Rename |
| space ca | Code action |
| space f | Format |

# Telescope
| Key      | Function |
| ----------- | ----------- |
| space ff | Find files |
| space fg | Live grep |
| space fb | Buffers |
| space fh | Search help |
